# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, and Gemini AI.

## Features

- ATS Resume Scoring
- Skill Gap Analysis
- AI Recruiter Insights
- Career Roadmap Generation
- Resume Parsing (PDF, DOCX, JPG, PNG)
- Modern Animated UI

## Tech Stack

- Python
- Streamlit
- Gemini AI
- NLP
- CSS

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
